# Full Stack Social Feed App

A full-stack social media style web application where users can create posts with images and captions, and display them dynamically in a feed.

---

## Features

- Create new posts
- Upload image with caption
- Display posts in feed
- Full-stack API integration
- MongoDB database connection
- Responsive frontend UI

---

## Tech Stack

### Frontend
- React
- Vite
- Axios
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

---

## Project Structure

```bash
FULLSTACKPROJ1

```

---

## Installation

### Clone Repository

```bash
git clone https://github.com/Navyajede/fullstackproj1.git
cd fullstackproj1
```

---

## Backend Setup

```bash
cd backend
npm install
npm start
```

Create `.env` file inside backend:

```env
MONGODB_URI=your_mongodb_connection_string
PORT=5000
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
